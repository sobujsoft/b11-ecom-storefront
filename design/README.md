# B11 Shop — Static Design Mockups

Standalone HTML design reference for the storefront. These files mirror the
Vue/Inertia pages in `resources/js/pages/shop/` but run with **no build step** —
open any `.html` file directly in a browser.

## Stack
- Tailwind CSS via Play CDN
- Lucide icons via CDN
- Shared chrome + mock data in `assets/shared.js`; mock orders in `assets/orders-data.js`

## Pages
| File | Page |
| --- | --- |
| `index.html` | Home (hero, categories, featured / new / popular) |
| `products.html` | Product listing with live filters & sorting |
| `product-detail.html` | Product detail, gallery, tabs, reviews (`?id=1`) |
| `cart.html` | Shopping cart + order summary |
| `wishlist.html` | Saved items |
| `checkout.html` | Contact, shipping, payment, order summary |
| `orders.html` | Order history |
| `order-detail.html` | Order detail with delivery timeline (`?id=1024`) |

## Notes
- Design-only: forms/buttons don't submit; data is static mock matching the API shapes.
- Header/footer and product cards are generated by `assets/shared.js` to avoid duplication.
- Navigation between pages works via the header and in-page links.
